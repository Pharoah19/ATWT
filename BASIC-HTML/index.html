CSS BASICS

  • Classes - Can be repeated 
                        .class-name{}
                
                        p class="class-name"
  • IDs - Can't be repeated, will work but it's invalid HTML to have multiple IDs with the same name. 
                        #id-name{}
                
                        p id="id-name"
  • CSS Properties: 
                        #id-name, .class-name {
                        property-name: value;
                    }
    Take a look at properties: CSS Properties
